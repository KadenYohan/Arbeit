FIGMA DESIGN PROMPT - ARBEIT PART-TIME JOBS PLATFORM
Below is the complete visual design specification for Figma. This includes every detail from Section 5 of the PRD.

PROJECT OVERVIEW
Create a web design system and key pages for Arbeit, a Metro Manila part-time jobs platform. The visual identity is inspired by:

SUUMO (Japan's property portal) — friendly green, information-dense, clean
NAVER (Korea's portal) — approachable, organized, interactive
Baitoru (Japanese job site) — compact cards, clear hierarchy, functional design

Design Philosophy:

Friendly and approachable — NOT cold or corporate
Information-dense but well-organized — NEVER cluttered
Interactive map as primary discovery feature — visual, not just text
Japanese compactness: every pixel earns its place
Accessible to all ages: large text, clear contrast, obvious CTAs
NO carousels that auto-play
NO modal popups on first load
NO decorative illustrations that add no information
